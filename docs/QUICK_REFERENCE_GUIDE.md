# CRM Quick Reference Guide

**One-Page Cheat Sheet for Daily Operations**

---

## Login & Access

**URL**: Your deployment URL
**First Login**: You'll be prompted to change your temporary password
**2FA**: If enabled, enter 6-digit code from authenticator app or email

---

## Quick Actions

**Keyboard Shortcuts**:
- `Ctrl+K` / `Cmd+K`: Open search
- `Ctrl+N` / `Cmd+N`: Quick add menu
- `Ctrl+/` / `Cmd+/`: Show shortcuts help
- `Esc`: Close any dialog

**Quick Add Button**: Click `➕` in bottom-right corner → Select entity type

---

## Creating Records

### Opportunity
1. Click `➕` → Opportunity
2. Fill: Company Name, Value, Stage, RE Type
3. Click Create

### Contact
1. Click `➕` → Contact
2. Fill: Full Name, Email, Role, Organization
3. Click Create

### Activity
1. Click `➕` → Activity
2. Select type (Call, Meeting, Note)
3. Fill: Summary, Related To
4. Click Create

---

## Opportunity Pipeline Stages

```
Prospect → Qualified → Proposal → Negotiation → Term Sheet → Won
   ↓          ↓          ↓            ↓            ↓         ↓
14 days   21 days    18 days      28 days      35 days   Project
```

**Each stage has Quality Gate checklist** - complete before advancing

---

## Dashboard Metrics

**Total Pipeline**: Sum of all open opportunities (excludes Lost)
**Active Deals**: Opportunities not in Won/Lost stage
**Active Projects**: Projects in Development/Construction/Operational
**Total Capacity**: Sum of all project capacities (MW)

## File Upload (Media Vault)

1. Open entity (Opportunity, Account, etc.)
2. Click "Documents" tab
3. Click "Upload File"
4. Select file → Choose category
5. Add description (optional)
6. Upload (GPS auto-captured on mobile)

**Categories**: Roof, Electrical, Utility Bill, Load Profile, Site Plan, Legal, Financial, Technical Report, Contract, Other

## Admin Functions

**User Management**: Admin Panel → Users → Create/Edit/Delete

**Activity Logs**: Admin Panel → Activity Logs (audit trail)

## Bulk Import

1. Navigate to entity list (e.g., Opportunities)
2. Click "Import" button
3. Download template Excel file
4. Fill template with your data
5. Upload completed file
6. Review column mapping & validation
7. Click "Import Now"


**Closed**: Archive status

## Troubleshooting

**Can't login?**
- Check email spelling
- Verify Caps Lock off
- Use "Forgot Password" link
- Contact admin if locked out

**Can't find record?**
- Check active filters
- Use search function
- Ask admin (may be owned by another user)

**Import failed?**
- Download error report
- Fix validation issues in file
- Re-upload corrected file

**Permission denied?**
- Confirm your role (Profile → View Role)
- Contact admin to adjust permissions

## Best Practices

✓ Log activities immediately (while details fresh)
✓ Complete Quality Gate checklists before advancing stages
✓ Use consistent company names (avoid duplicates)
✓ Enable 2FA on your account
✓ Tag contacts with decision-making roles
✓ Link opportunities to accounts (single source of truth)
✓ Review Upcoming Actions daily
✓ Update opportunity stages weekly

## Quick Stats Formulas

**Win Rate** = (Won Opportunities / Total Opportunities) × 100%
**Average Deal Size** = Total Pipeline Value / Number of Deals
**Conversion Rate** = (Stage N+1 Count / Stage N Count) × 100%
**Cycle Time** = Average days from Prospect to Won
